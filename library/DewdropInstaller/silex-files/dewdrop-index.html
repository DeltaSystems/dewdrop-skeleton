Common Next Steps

Install Bower components

Dewdrop uses Bower to manage client-side components. Install them by running bower install in your project's folder.

Setting up user authentication

If you plan on using the stock Auth components from Dewdrop, check out the API docs for them to get started.

Using the default admin shell

The admin shell is already installed and configured. Head to the example component to get started.
